# Materials and Analysis Code for CHI Paper

This repository contains the data, R scripts, and study materials used in the analyses
reported in the CHI paper:

_Robin Bretin, Robin Dheilly, and Jessica Cauchard. 2026. ENtry, Occupancy, EXit (NOX): A Model of Human–Robot Territorial Dynamics. In Proceedings of the 2026 CHI Conference on Human Factors in Computing Systems (CHI’26), April 13–17, 2026, Barcelona, Spain. ACM, New York, NY, USA, 32 pages. https://doi.org/10.1145/3772318.3791257_

## Important Note

These materials are shared in the state used by the authors and are not fully cleaned,
documented, or packaged for reuse. Some scripts may depend on local file paths and
intermediate files. Variable names may not be self-explanatory.

They are provided to support transparency and inspection of the analyses rather than as
a fully reproducible pipeline.

## Questions

If you have questions about the materials or analyses, feel free to contact the authors.

## Contents

- /data
  - data_pretest.csv: dataset from pre-test study
  - data_main.csv: dataset from main study

- /code
  - analysis_pretest.R: R scripts used for pre-test analyses
  - analysis_main.R: R scripts used for main study analyses

- /figures: figures generated during analysis and used in the paper
